## Overview

The applications uses Mantine-v7 library to display the tables from the .json file.


## Installation

1. **Clone the Repository**
    ```bash
    git clone https://github.com/radorification/tasko
    cd tasko
    ```

2. **Install Dependencies**
    ```bash
    npm install
    ```

3. **Run the Application**
    ```bash
    npm run dev
    ```
